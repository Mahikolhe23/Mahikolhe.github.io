# Portfolio Website

Single-file GitHub Pages portfolio for Mahendra Kolhe.

Live site: [mahikolhe23.github.io/Mahikolhe.github.io](https://mahikolhe23.github.io/Mahikolhe.github.io/)

## Purpose

This repository is used to deploy the portfolio website through GitHub Pages. The main professional introduction stays in the GitHub profile README, and the full portfolio content lives in `index.html`.

## Tech

- Single `index.html`
- Tailwind CSS via CDN
- Inline JavaScript
- GitHub Actions Pages deployment

## Update Flow

1. Edit `index.html`.
2. Push changes to `main`.
3. GitHub Pages deploys automatically through `.github/workflows/pages.yml`.

The `dev` branch is used for working changes before publishing to `main`.
